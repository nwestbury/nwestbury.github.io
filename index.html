investleduc.com

Along with a friend, I helped develop investleduc.com. I worked particularly with the backend. Tasks included: autofilling and emailing a generated PDF, creating an email server, creating an email subscription service, and creating an administrator panel.

Curling Project

In the summer of 2015, I had the unique opportunity to work in Michael Bowling's (a famous artificial intelligence professor) in his curling lab. I worked on improving the graphical interface of the simulator, improving some of the curling "bots" using logistic and linear machine learning methods, and running (a lot) of experiments (some on a supercomputer!).

  • For a little more information you can download my poster ("Richardson" is the name of our best curling bot): Curling Poster PDF

robotpoet.com

More of a fun project, robotpoet.com tries to find relevant rhymes to a given word, highlighting them on perceived relevance. The backend was quite challenging given the over 1GB of raw data we have. Initially, we used a MySQL database but performance was too slow so we switch to using a C script that uses precomputed binary files to return the results instead. Read more about it there.
